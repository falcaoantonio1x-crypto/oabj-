# OAB — Plataforma (IndexedDB + SRS) — com 400 questões
- Banco: `data/questoes.json` (já incluído)
- Persistência: IndexedDB (OABDatabase)
- Progresso: cards SRS (revisão por vencimento) + tempo de resposta
- Feedback: sempre comenta (certo/errado) + base legal + exemplo + vídeo

## Publicar no GitHub Pages
Sobe a pasta toda na raiz do repo e ativa Pages em `main` + `/ (root)`.

## Se trocar o banco
1) Substitui `data/questoes.json`
2) No site → aba Admin → “Recarregar”
